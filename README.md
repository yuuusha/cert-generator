# Генератор грамот и сертификатов

Программный продукт **«Генератор грамот и сертификатов»** предназначен для автоматизации создания наградных документов по заранее подготовленным шаблонам.

Система позволяет:

- автоматически подставлять данные участников в шаблоны документов;
- генерировать персонализированные грамоты, сертификаты и дипломы;
- сохранять результаты генерации;
- вести историю операций;
- использовать базу данных для хранения мероприятий, участников, шаблонов и результатов работы;
- работать через удобный графический интерфейс.

Приложение реализовано как десктопная Java-система с использованием:

- Java 21
- Spring Boot
- Swing GUI
- PostgreSQL
- JDBI
- Flyway
- PDFBox

---

# Основные возможности

## Работа с мероприятиями

Система позволяет:

- создавать мероприятия;
- автоматически создавать директорию для хранения результатов генерации;
- использовать мероприятия при генерации документов.

---

## Работа с участниками

Система поддерживает:

- хранение ФИО;
- хранение университета;
- хранение города;
- хранение факультета;
- выбор участника из выпадающего списка.

---

## Работа с шаблонами

Поддерживаются шаблоны следующих типов:

- PNG
- JPEG
- PDF
- DOCX

Шаблоны регистрируются в системе и сохраняются в базе данных.

---

## Генерация документов

Поддерживаются типы документов:

- Грамота (`GRAMOTA`)
- Сертификат (`CERTIFICATE`)
- Диплом (`DIPLOMA`)

Поддерживаемые выходные форматы:

- PDF
- PNG

Во время генерации:

- данные участника автоматически наносятся на шаблон;
- текст масштабируется под размеры области;
- создается итоговый файл;
- сохраняется запись в истории генерации.

# Настройка проекта

## 1. Клонирование проекта

```bash
git clone <repository-url>
cd cert-generator
```

---

## 2. Настройка переменных окружения

Создать файл `.env`:

```env
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=

POSTGRES_URL=
FLYWAY_URL=
```

---

## 3. Запуск PostgreSQL

```bash
docker-compose up -d
```

---

## 4. Выполнение миграций

Flyway выполнит миграции автоматически при запуске контейнеров.

---

## 5. Запуск приложения

```bash
./gradlew bootRun
```

или

```bash
gradlew.bat bootRun
```

---

# Пример сценария использования

## 1. Добавление мероприятия

Пользователь создает мероприятие:

```text
Научная конференция 2026
```

---

## 2. Добавление участника

Пользователь добавляет:

```text
ФИО: Иванов Иван Иванович
ВУЗ: МГУ
Город: Москва
Факультет: ФКН
```

---

## 3. Добавление шаблона

Пользователь выбирает файл:

```text
certificate_template.png
```

---

## 4. Генерация

Пользователь выбирает:

- мероприятие;
- участника;
- шаблон;
- тип документа;
- формат;

и нажимает кнопку:

```text
Сгенерировать
```

После этого создается итоговый документ.
